# williamcruz.ch

My personal slice of the internet provides a platform for my writing and to showcase my latest work.

## Overview

- `pages/*` - All other static pages.

## Running Locally

```bash
$ git clone https://github.com/cruzentwickler/myportfolio.git
$ cd myportfolio-main
$ yarn
$ yarn dev
```
